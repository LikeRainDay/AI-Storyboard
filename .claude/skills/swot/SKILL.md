# SWOT 战略分析顾问 📊

## SWOT 分析框架

SWOT 模型包含四个核心维度：

### 💪 0.1 优势（Strengths）

**内部因素 - 正面**

分析企业的核心竞争力：

- 品牌价值和市场地位
- 技术创新和研发能力
- 人才团队和组织能力
- 财务状况和资本实力
- 供应链和运营效率
- 产品和服务质量
- 客户基础和忠诚度

### ⚠️ 0.2 劣势（Weaknesses）

**内部因素 - 负面**

识别企业的短板和不足：

- 成本结构和盈利能力
- 管理体系和流程效率
- 技术差距和创新不足
- 人才流失和团队稳定性
- 品牌认知度和影响力
- 市场份额和覆盖范围
- 产品线和服务缺陷

### 🌟 0.3 机会（Opportunities）

**外部因素 - 正面**

把握市场和环境机遇：

- 市场增长和需求趋势
- 政策支持和行业利好
- 技术变革和创新机会
- 新兴市场和客户群体
- 战略合作和并购机会
- 国际化和区域扩张
- 产业链整合机会

### 🚨 0.4 威胁（Threats）

**外部因素 - 负面**

识别潜在风险和挑战：

- 竞争加剧和价格战
- 政策变化和监管风险
- 技术颠覆和替代品
- 经济周期和市场波动
- 供应链风险和成本上升
- 客户需求变化
- 地缘政治和贸易风险

## 分析方法论

### 📊 数据驱动

- 使用最新的行业报告和财务数据
- 引用权威机构和分析师观点
- 提供量化指标和对比数据

### 🔍 多维度视角

- 从战略、运营、财务、市场等多角度分析
- 结合行业趋势和竞争格局
- 考虑短期和长期影响

### 💡 可执行建议

- 基于 SWOT 分析提出战略建议
- SO 策略：利用优势抓住机会
- WO 策略：克服劣势把握机会
- ST 策略：利用优势应对威胁
- WT 策略：减少劣势和威胁

## 输出质量保证

每份分析报告将包含：

✅ **全面性**：覆盖 SWOT 四个维度的详细分析
✅ **准确性**：基于最新数据和权威信息源
✅ **深度性**：不仅列举要点，还提供深入解读
✅ **实用性**：提供可执行的战略建议
✅ **结构化**：清晰的章节和逻辑组织
